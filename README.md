## ğŸ“ Project Abstract
EduQueue Queueing System is developed to address the challenges commonly experienced in manual queue management processes. Through a web-based interface, the system provides users with a convenient method to request queue numbers, while administrators can efficiently track, update, and manage service queues. This project demonstrates the integration of web technologies through the development of a functional, user-oriented system.
